# 📇 SmartContact Manager

A full-stack contact management web application that lets users securely add, organize, and share their contacts through a clean, responsive interface.

## ✨ Features
- User authentication & authorization with Spring Security
- Add, edit, delete, and search contacts
- Email notifications via integrated Email API
- Responsive front-end built with Thymeleaf, HTML, CSS & JavaScript

## 🛠️ Tech Stack
`Spring Boot` · `Spring Security` · `Thymeleaf` · `JavaScript` · `HTML/CSS` · `MySQL`

## 🚀 Getting Started
1. Clone the repo
2. Configure your MySQL database in `application.properties`
3. Run `mvn spring-boot:run`
4. Visit `http://localhost:8080`
